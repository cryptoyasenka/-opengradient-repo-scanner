# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Email:** [cryptoyasenka@users.noreply.github.com](mailto:cryptoyasenka@users.noreply.github.com)

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

I will respond within 48 hours and work on a fix as quickly as possible.

## Scope

This policy applies to the latest version of the code on the `main` branch.

## TEE Verification

All AI analysis results are cryptographically verified via OpenGradient's Trusted Execution Environment. On-chain proofs are recorded on Base Sepolia and cannot be retroactively altered.
